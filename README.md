Когда пользователь открывает эту страницу, он попадает в раздел «Дневник» сайта хирургического центра ортопедии и трансплантологии. Вверху виден заголовок «Дневник», а под ним — навигационное меню со ссылками на все основные разделы: «Логин», «Мой план», «Упражнения», «Дневник», «Врачи» и «Справка».

Далее отображается заголовок «Дневник самочувствия» и форма для записи данных о самочувствии. В форме пользователь может:
указать дату и время с помощью поля типа datetime-local;

оценить уровень боли по шкале от 1 до 10 — для этого есть ползунок, изначально установленный на значении 3;

выбрать уровень отёчности из выпадающего списка (варианты: «Отсутствует», «Лёгкая», «Умеренная», «Сильная»);/
ввести текстовое описание самочувствия в поле «Описать самочувствие».
После заполнения всех нужных полей пользователь нажимает кнопку «Сохранить». При этом:
отправка формы перехватывается JavaScript кодом (через e.preventDefault()), поэтому страница не перезагружается;
появляется всплывающее окно с сообщением «Запись сохранилась»;
форма автоматически очищается (благодаря this.reset()), и пользователь может сразу внести новую запись.
